## Getting Started

First, run the development server:

```bash
npm install
pnpm start:dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Commands

- `start:dev`: runs your application on `localhost:3000`
- `lint`: runs the linter in all project
